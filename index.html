Featured Project

Chi-Chi is SQC's signature robotics build, designed as an agile and expressive robotic dog platform.

Robotics Flagship

Chi-Chi: SQC Robotic Dog

Chi-Chi represents the club's robotics vision: a dog-inspired robot that combines mechanical design, electronics, embedded programming, and control logic in one complete system.

The project is used as a live learning platform where members collaborate on gait experiments, balance tuning, motion control, and behavior-driven interactions.

  • Locomotion and movement control prototypes
  • Sensor integration and real-time feedback loops
  • Collaborative hardware plus software development
  • Showcase-ready demonstration for tech events
